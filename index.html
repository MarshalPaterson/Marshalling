Marshalling

Marshalling rounds up your data from client to server.

Welcome to Marshalling.

Marshalling is a Typescript framework that makes it easy to handle data and data manipulation to a UI from a web service response.

In a nutshell Marshalling is a dispatch and listerner framework using UI, Commands, Delegates, Services and Events.
